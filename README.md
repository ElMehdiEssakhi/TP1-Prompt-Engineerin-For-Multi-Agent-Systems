# LangChain Project

This project demonstrates how to use **LangChain** with OpenAI APIs and other tools. It includes instructions for setup, dependency installation, and environment configuration.

---

## Prerequisites

- Python 3.11+
- `uv` (or `poetry`) for dependency management
- An OpenAI API key (or other LLM API keys)

---

## Installation

1. **Clone the repository**:

```bash 
git clone https://github.com/ElMehdiEssakhi/TP1-Prompt-Engineerin-For-Multi-Agent-Systems
cd TP1-Prompt-Engineerin-For-Multi-Agent-Systems
```

2. **Install dependencies using uv**:

```bash
uv install
```
3. **Configure API Keys**:
Create a .env file in the project root:

```dotenv
OPENAI_API_KEY=your_openai_api_key_here
